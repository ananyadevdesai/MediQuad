# MediQuad

A compact modular quadruped robot for healthcare assistance and patient monitoring.

## Overview

MediQuad is an open-source healthcare robotics project designed to bring autonomous assistance to medical environments. This repository contains the project documentation website.

## Features

- Autonomous navigation in hospital environments
- Patient monitoring capabilities
- Medical supply delivery
- Voice interaction for patient communication
- Thermal imaging for temperature screening
- Modular design with 12 independent servo motors

## Local Development

To view the website locally:

1. Clone this repository
2. Open `index.html` in your browser, or serve the folder:

```bash
python3 -m http.server 5173
```

3. Navigate to `http://localhost:5173`
