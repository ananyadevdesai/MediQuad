# Assets Directory

Place your MediQuad images here:

## Required Images

- `MediQuad-1.jpeg` - Printed parts photo
- `MediQuad-2.jpeg` - Assembly layout diagram
- `MediQuad-3.jpeg` - CAD side view
- `MediQuad-4.jpeg` - CAD front view
- `mediquad-visualization.jpeg` - System visualization and diagnostics

## Image Specifications

- Format: JPEG or PNG
- Recommended size: 1200x1200px for gallery images
- Aspect ratio: 1:1 for gallery, 16:9 for visualization

## Note

These images are referenced in the HTML files. Make sure to add your actual images here before deploying the website.
